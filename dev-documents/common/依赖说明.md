# devDependencies
- `typescript`: TS支持
- `vite`: 构建工具
- `ts-jest` + `jest` + `@types/jest`: 支持TS的单元测试
- `jest-canvas-mock`: 为 `jest` 测试 `Canvas` 提供支持
- `vite-plugin-dts`: `vite`插件，用以库模式打包时，导出`d.ts`类型声明文件

# dependencies
- `jimp`: JS 图像处理库 [JIMP](https://github.com/oliver-moran/jimp)
